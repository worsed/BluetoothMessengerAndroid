# 📱 Как установить APK на Android

## Способ 1: Прямая установка (САМЫЙ ПРОСТОЙ)

### Шаг 1: Скачайте APK

1. Откройте https://github.com/worsed/BluetoothMessengerAndroid
2. Вкладка "Actions"
3. Кликните на последнюю зеленую галочку
4. Прокрутите вниз до "Artifacts"
5. Скачайте "BluetoothMessenger-Debug"
6. Распакуйте ZIP файл
7. Найдите файл `app-debug.apk`

### Шаг 2: Разрешите установку

На Android:
1. Настройки → Безопасность
2. Найдите "Неизвестные источники" или "Установка из неизвестных источников"
3. Разрешите для вашего браузера/файлового менеджера

Или:
1. При попытке установки появится запрос
2. Нажмите "Настройки"
3. Включите переключатель
4. Вернитесь назад

### Шаг 3: Установите APK

1. Откройте файл `app-debug.apk` на телефоне
2. Нажмите "Установить"
3. Дождитесь завершения
4. Нажмите "Открыть"

### Шаг 4: Разрешите доступ

При первом запуске приложение запросит:
- ✅ Bluetooth - разрешите
- ✅ Местоположение - разрешите (нужно для поиска устройств)
- ✅ Уведомления - разрешите (опционально)

## Способ 2: Через ADB (для разработчиков)

```bash
# Подключите телефон к компьютеру
# Включите "Отладка по USB" в настройках разработчика

# Установите APK
adb install app-debug.apk

# Или если уже установлено (переустановка)
adb install -r app-debug.apk
```

## Способ 3: Через Android Studio

1. Откройте проект в Android Studio
2. Подключите телефон USB кабелем
3. Нажмите Run (зеленая стрелка)
4. Выберите ваше устройство
5. Приложение установится автоматически

## ⚠️ Важно

### Безопасность

- APK не подписан официальным сертификатом
- Это нормально для debug версии
- Для production нужна подпись

### Разрешения

Приложение запрашивает:
- **Bluetooth** - для связи между устройствами
- **Местоположение** - Android требует для поиска Bluetooth устройств
- **Уведомления** - для уведомлений о новых сообщениях

### Совместимость

- ✅ Android 8.0 (Oreo) и выше
- ✅ Работает на всех Android устройствах
- ❌ НЕ работает с iPhone

## 🔧 Решение проблем

### "Приложение не установлено"

→ Удалите старую версию и попробуйте снова

### "Установка заблокирована"

→ Разрешите установку из неизвестных источников

### "Приложение не запускается"

→ Проверьте что Android 8.0 или выше

### "Не находит устройства"

→ Разрешите доступ к местоположению
