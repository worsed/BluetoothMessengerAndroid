# 📱 Bluetooth Messenger для Android

Мессенджер для обмена сообщениями между Android устройствами через Bluetooth.

## ✨ Возможности

- 📡 Автоматическое обнаружение устройств поблизости
- 💬 Обмен текстовыми сообщениями в реальном времени
- 💾 Сохранение истории сообщений
- 🔔 Уведомления о новых сообщениях
- 🎨 Современный Material Design 3 интерфейс
- 🌙 Темная и светлая темы
- 🔄 Автоматическое переподключение

## 📥 Установка

### Способ 1: Скачать готовый APK (САМЫЙ ПРОСТОЙ)

1. Перейдите на https://github.com/ваш_username/BluetoothMessengerAndroid
2. Вкладка "Actions"
3. Выберите последнюю успешную сборку
4. Скачайте "BluetoothMessenger-Debug.apk" из Artifacts
5. Откройте APK на телефоне
6. Разрешите установку из неизвестных источников
7. Установите приложение

### Способ 2: Собрать самостоятельно

```bash
# Клонируйте репозиторий
git clone https://github.com/ваш_username/BluetoothMessengerAndroid.git
cd BluetoothMessengerAndroid

# Соберите APK
./gradlew assembleDebug

# APK будет в app/build/outputs/apk/debug/
```

## 🚀 Использование

1. Запустите приложение на двух Android устройствах
2. Разрешите доступ к Bluetooth и местоположению
3. На одном устройстве откройте Настройки → Обновить список
4. Выберите устройство для подключения
5. Начните обмен сообщениями!

## 🔧 Требования

- Android 8.0 (API 26) или выше
- Bluetooth
- Разрешения:
  - Bluetooth
  - Местоположение (для обнаружения устройств)
  - Уведомления

## 🛠️ Технологии

- **Kotlin** - язык программирования
- **Jetpack Compose** - современный UI фреймворк
- **Material Design 3** - дизайн система
- **Bluetooth Classic** - связь между устройствами
- **Coroutines** - асинхронность
- **ViewModel** - архитектура

## ⚙️ Разработка

### Структура проекта

```
BluetoothMessengerAndroid/
├── app/
│   ├── src/main/
│   │   ├── java/com/btmessenger/android/
│   │   │   ├── MainActivity.kt           # Главная активность

│   │   │   ├── bluetooth/
│   │   │   │   └── BluetoothService.kt   # Bluetooth логика

│   │   │   │   └── MessengerViewModel.kt # Бизнес-логика
│   │   │   ├── data/

│   │   │   └── ui/theme/

│   │   ├── res/                          # Ресурсы
│   │   └── AndroidManifest.xml           # Манифест
│   └── build.gradle.kts                  # Конфигурация сборки
├── build.gradle.kts                      # Корневая конфигурация
└── settings.gradle.kts                   # Настройки проекта
```

### Сборка

```bash
# Debug версия
./gradlew assembleDebug

# Release версия
./gradlew assembleRelease

# Установка на подключенное устройство
./gradlew installDebug
```

## 🐛 Известные проблемы

- На Android 12+ требуется разрешение BLUETOOTH_CONNECT
- Обнаружение устройств требует включенного местоположения
- Радиус действия зависит от устройства (обычно 10-30 метров)
